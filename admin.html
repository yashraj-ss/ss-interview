Setup — takes 5 minutes, done once forever
Uses GitHub to store data. Free, permanent, no credit card.
1
Create a GitHub account (if you don't have one)
Go to github.com/signup and create a free account. If you already have one, skip this step.
2
Create a new repository
Go to github.com/new
· Repository name: ss-interview
· Set it to Public
· Tick "Add a README file"
· Click Create repository
3
Enable GitHub Pages
In your new repo → click Settings (top menu) → scroll down to Pages (left sidebar)
· Under "Branch" — select main from the dropdown → click Save
· Wait 2 minutes. Your site will be live at https://YOUR-USERNAME.github.io/ss-interview/
4
Create a Personal Access Token
Go to github.com/settings/tokens/new
· Note: ss-interview-token
· Expiration: No expiration
· Scopes: tick repo (the top checkbox — this selects everything under it)
· Scroll down → click Generate token
· Copy the token now — you won't see it again
5
Upload the portal files to your repo
In your repo on GitHub → click Add file → Upload files
· Drag all 4 files: admin.html, candidate.html, submissions.json, config.json
· Click Commit changes
6
Enter your GitHub details below
Fill in the three fields and click Save & Test. If it says Connected, you're done.
